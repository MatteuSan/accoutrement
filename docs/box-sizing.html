OddBird's Sass Accoutrement » box-sizing

@mixin global-box-sizing()

Establish gloabl box-sizing within any scope.

Parameters

$box: border-box (border-box | content-box)

The box-sizing model to establish globally.

Examples

scss 
@include global-box-sizing;
scss – nested box-sizing 
.old-box-model {
  box-sizing: content-box;
  @include global-box-sizing(inherit);
}
